What I'm learning

A bit about my current program

My current program is the Business Technology Management (BTM) program. This program teaches about the fundamentals of business, technology and management and the skills relevant in pursuit of rapidly growing career paths in those fields, such as Business Intelligence and Information Technology.

My favourite courses

Among the various courses I took so far in BTM, here are my top 3:

  1. ITM618: Business Intelligence and Analytics

    This course put my Python coding skills to the test with the use of the Pandas library to find insights in a real data set. As someone looking into pursuing a career in Data Analytics, I believe that the skills I gained will be extremely relevant in my future endeavours, making this one of the most useful courses I took so far.

  2. ITM711: Cloud Computing

    This course was about both the theoretical and practical applications of cloud computing. I really enjoyed doing the labs in this course, which included deploying websites and virtual machines using Amazon Web Services.

  3. ITM301: IT Infrastructure

    This course was an introductory course about computer hardware/software and communication networks. I found this course very interesting as someone fascinated by computer hardware. The software part of it was very practical too.
